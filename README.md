## 主要修改
增加大小号功能，主要是防止未实名小号施肥，影响种果树。
不通知赶鸡名单、不复活能量名单改成通知赶鸡名单复活能量名单。
## 使用方法
切换到每个号都重新设置并保存。
## 其它说明
功能上没有任何新增/修改/删除，也未修补任何（明面上or潜在的）bug。
